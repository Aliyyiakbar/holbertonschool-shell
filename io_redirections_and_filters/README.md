# Holberton School - Shell, I/O Redirections and filters

This directory contains scripts that practice redirections, pipes, and classic command-line filters.

## Requirements

- OS: Ubuntu 20.04 LTS
- All scripts start with: #!/bin/bash
- All scripts are exactly 2 lines long
- No backticks, &&, ||, or ;
- No sed or awk
- All scripts are executable

## Tasks

- 0. Hello World — print “Hello, World”
- 1. Confused smiley — print a confused smiley
- 2. Let's display a file — display the contents of a file
- 3. What about 2? — display the contents of two files
- 4. Last lines of a file — display the last 10 lines of a file
- 5. I'd prefer the first ones actually — display the first 10 lines of a file
- 6. Line #2 — display the third line of a file
- 7. It is a good file that cuts iron without making a noise — create a file with specific content
- 8. Save current state of directory — write the result of ls into a file
- 9. Duplicate last line — duplicate the last line of a file
- 10. No more javascript — delete .js files (recursively)
- 11. Don't just count your directories, make your directories count — count directories and subdirectories
- 12. What’s new — display newest files
- 13. Being unique is better than being perfect — print unique lines
- 14. It must be in that file — find a word in a file
- 15. Count that word — count occurrences of a word in a file
- 16. What's next? — display lines containing a pattern and the 3 lines after them
- 17. I hate bins — display all lines that do not contain a pattern
- 18. Letters only please — display lines starting with letters only
- 19. A to Z — translate characters (custom mapping)
- 20. Without C, you would live in hiago — remove letters c and C
- 21. esreveR — reverse input
- 22. DJ Cut Killer — display users and their home directories sorted by username
- 23. Empty casks — find all empty files and directories
- 24. A gif is worth ten thousand words — list .gif files without extensions
- 25. Acrostic — decode acrostics using first letters of each line
- 26. The biggest fan — list the 11 most active hosts or IPs from TSV logs

## Usage

```bash
chmod u+x <script_name>
./<script_name>
```
