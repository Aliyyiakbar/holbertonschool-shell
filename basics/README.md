# Shell, basics

This directory contains scripts for learning the shell basics: navigation, listing files,
manipulating files/directories, and working with links and wildcards.

## Requirements

- OS: Ubuntu 22.04 LTS
- All scripts start with: #!/bin/bash
- All scripts are exactly 2 lines long
- No backticks, &&, ||, or ;
- All scripts are executable

## Tasks

- 0. Where am I? — print the absolute path of the current working directory
- 1. What’s in there? — list files in the current directory
- 2. There is no place like home — change directory to the home directory
- 3. The long format — list files in long format
- 4. Hidden files — list all files including hidden ones
- 5. I love numbers — list files with numeric user/group IDs
- 6. Welcome — create a directory in /tmp
- 7. Betty in my first directory — move a file into a directory
- 8. Bye bye Betty — delete a file
- 9. Bye bye My first directory — delete a directory
- 10. Back to the future — change to the previous directory
- 11. Lists — list files in current, parent, and /boot directories (long format)
- 12. File type — print the type of a file
- 13. We are symbols, and inhabit symbols — create a symbolic link
- 14. Copy HTML files — copy HTML files that are newer or missing in destination
- 15. Let’s move — move files that start with an uppercase letter
- 16. Clean Emacs — delete Emacs backup files (*~)
- 17. Tree — create a directory tree structure

## Usage

```bash
chmod u+x <script_name>
./<script_name>
```
