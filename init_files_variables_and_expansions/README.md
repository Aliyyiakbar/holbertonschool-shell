# Shell, init files, variables and expansions

This directory contains scripts about aliases, variables, expansions, and shell arithmetic.

## Requirements

- OS: Ubuntu 20.04 LTS
- All scripts start with: #!/bin/bash
- All scripts are exactly 2 lines long
- No &&, ||, or ;
- No bc, sed, or awk
- All scripts are executable

## Tasks

- 0. <o> — create an alias
- 1. Hello you — print hello + current user
- 2. The path to success is to take massive, determined action — add a directory to PATH
- 3. If the path be beautiful, let us not ask where it leads — count directories in PATH
- 4. Global variables — list environment variables
- 5. Local variables — list local variables, environment variables, and functions
- 6. Local variable — create a new local variable
- 7. Global variable — create a new global variable
- 8. Every addition to true knowledge is an addition to human power — print result of an addition
- 9. Divide and rule — print result of a division
- 10. Love is anterior to life, posterior to death, initial of creation, and the exponent of breath — exponent calculation
- 11. There are 10 types of people in the world -- Those who understand binary, and those who don't — convert binary to decimal
- 12. Combination — print combinations of two digits
- 13. Floats — print a number with two decimal places
- 14. Decimal to Hexadecimal — convert decimal to hexadecimal
- 15. What happens when you type ls *.c (Advanced) — wildcard expansion exercise
- 16. Everyone is a proponent of strong encryption (Advanced) — encode/decode with ROT13
- 17. The eggs of the brood need to be an odd number (Advanced) — print every other line from input
- 18. I'm an instant star. Just add water and stir. (Advanced) — add two numbers stored in variables

## Usage

```bash
chmod u+x <script_name>
./<script_name>
```
